# Configuration UI

## Propósito
Definir la interfaz de configuración (YAML-driven): carga de `config_master.yaml`, edición de parámetros y guardado de overrides por escenario.

## Funcionalidades
- Cargar `config_master.yaml` en árbol
- Editar parámetros numeric/text
- Guardar overrides per-scenario en `overrides/` o en `config_master` overrides
