# 07_SOFTWARE_ARCHITECTURE/README

Propósito: Documentar la arquitectura software prevista para el `ingetrans-reel-simulator` V4.

Contenido:
- `01_modules_overview.md` - estructura modular del sistema
- `02_classes.md` - clases principales, atributos y métodos
- `03_interfaces.md` - interfaces públicas y contratos
- `04_api_rest.md` - diseño de la API REST
- `05_database_schema.sql` - esquema SQL para persistencia
- `06_simulation_engine.md` - diseño del motor de simulación
- `07_animation_engine.md` - diseño del motor de animación (trazas)
- `08_dashboard.md` - arquitectura del dashboard y endpoints
- `09_reports.md` - plantillas y pipeline de reports
- `10_configuration_ui.md` - diseño de la UI de configuración (YAML driven)
- `11_exports.md` - formatos y endpoints de exportación

Prioridad: Alta. Este módulo define contratos y límites entre motores de simulación, persistencia y visualización.
