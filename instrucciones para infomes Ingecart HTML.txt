Aquí tienes un comando estructurado en formato JSON, diseñado para ser ejecutado por un agente de IA o un asistente automatizado dentro de un IS Backoffice (Sistema de Backoffice Inteligente).

Este comando ordena al agente identificar todos los informes pendientes, extraer sus datos y renderizarlos en HTML usando exactamente el libro de estilo Dark Industrial Premium de INGECART (negro, naranja, grises) y generar versiones simultáneas en castellano (es) e inglés (en).

Comando para el Agente
json

  "agent_command": "generate_bilingual_html_reports",

    "data_source": {
      "type": "internal_database",
      "query": "SELECT * FROM pending_reports WHERE status = 'ready_to_generate' ORDER BY priority DESC",

    },
    "output": {
      "base_directory": "/var/www/reports/",
      "structure": {
        "es": "{report_id}/index_es.html",
        "en": "{report_id}/index_en.html"
      },
      "overwrite_existing": true

Ejemplo de uso en consola (si tu agente admite CLI)
Si tu agente tiene una interfaz de línea de comandos, podrías lanzarlo así:

bash
agent-cli run --command generate_bilingual_html_reports --config ./agent_config.json --data ./reports_data.json
¿Qué hace este comando en la práctica?
Escanea todos los informes pendientes en tu Backoffice.

Inyecta los datos en la misma estructura HTML que te entregué en el mensaje anterior (cabecera, hero, tablas con diseño oscuro, sidebar de contacto, footer).

Duplica el contenido en ambos idiomas sin depender de conexión a internet para la traducción (el agente usa su capacidad lingüística interna).

Garantiza que todos los informes de la organización (ventas, producción, calidad, mantenimiento, etc.) compartan la misma identidad visual premium, consolidando la marca INGECART en todos los canales.
