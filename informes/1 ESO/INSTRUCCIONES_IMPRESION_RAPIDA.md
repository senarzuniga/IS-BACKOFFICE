# INSTRUCCIONES DE IMPRESIÓN

El archivo REPASO_COMPLETO_IMPRIMIBLE.md contiene toda la gramática, vocabulario y ejercicios por temas, listo para imprimir o convertir a PDF/Word.

1. Abre REPASO_COMPLETO_IMPRIMIBLE.md en tu editor favorito (VS Code, Typora, etc.).
2. Si quieres PDF, imprime desde el editor o usa la opción "Exportar como PDF".
3. Si quieres Word, copia y pega el contenido en Word y dale formato si lo deseas.
4. Imprime y úsalo como documento de trabajo y repaso.
