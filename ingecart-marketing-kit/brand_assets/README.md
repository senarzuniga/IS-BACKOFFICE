## Usage Instructions

For co-branded materials (e.g., FESPA stand banners, joint pitch decks):
1. Use Ingecart logo on the right / secondary position
2. Use IAR logo on the left / primary position
3. Apply the label: "Powered by IAR AI Technology" beneath the Ingecart logo
4. Maintain a visual ratio of approximately 60% (Ingecart) / 40% (IAR)
