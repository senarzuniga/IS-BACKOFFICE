# Instrucciones de uso

1. Abre el archivo scripts/leads_fespa2026_enriched.xlsx para consultar los leads con información enriquecida y contexto de empresa.
2. Abre el archivo scripts/leads_fespa2026_acciones.xlsx para ver las acciones recomendadas, priorizadas y listas para ejecutar.
3. Puedes compartir ambos archivos con el equipo comercial para su seguimiento y ejecución.

Si necesitas actualizar o volver a generar los archivos, ejecuta los scripts en la carpeta scripts/:
- enrich_leads.py
- generate_leads_actions.py
