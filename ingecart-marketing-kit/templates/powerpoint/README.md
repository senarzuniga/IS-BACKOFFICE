# PowerPoint Patterns - Ingecart

This folder stores reusable python-pptx patterns for Ingecart slides.

## Available pattern
- `ingetrans_slide_pattern.py`: Generates the corporate Ingetrans ad slide.

## How to generate
From repo root:

```powershell
python ingecart-marketing-kit/templates/powerpoint/ingetrans_slide_pattern.py
```

Output file:
- `ingecart-marketing-kit/assets/presentations/Ingecart_Corporate_Slide.pptx`

## Reuse in future
- Duplicate the pattern file for a new slide type.
- Keep shared style constants (colors, spacing, typography) aligned with this baseline.
- Replace only text blocks and optional shape/image sections.
