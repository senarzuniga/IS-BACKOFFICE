# Plan de análisis y enriquecimiento de leads FESPA 2026

1. Extraer datos del Excel original y convertir a formato procesable.
2. Analizar y completar información faltante de cada lead (enriquecimiento web, búsqueda de contexto de empresa, etc.).
3. Generar nuevo Excel con campos completos y columna "Company Context".
4. Crear Excel de acciones priorizadas para cada lead, siguiendo la plantilla de generación de acciones.
5. Validar que las acciones sean ejecutables y contengan toda la información necesaria.
6. Entregar ambos archivos generados.
