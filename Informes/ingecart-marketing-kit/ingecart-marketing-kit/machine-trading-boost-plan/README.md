# Ingecart Machine Trading Boost Plan (Global)

## Objective
Launch and scale Ingecart's business line for:
- Trade and relocation of used fruit packing machines (3 units ready to sell)
- Trade and relocation of 1 Flexo Folder Gluer (FFG)
- Service layer: audit, dismantling, relocation, installation, commissioning, training, and after-sales support

This package is designed as an execution folder (not only analysis).

## Folder Contents
- `01_market_analysis_worldwide.md`: market dynamics, demand signals, risk/opportunity map
- `02_operator_map_and_companies.md`: ecosystem map and priority company universe
- `03_icp_and_prospecting.md`: ICP, segmentation, target-account logic, and lead scoring
- `04_outreach_playbooks.md`: practical multi-channel outbound strategy and scripts
- `05_demand_discovery_and_qualification.md`: how to discover real demand and qualify opportunities
- `06_events_hubs_sites.md`: sector sites, events, associations, and global hubs
- `07_scraping_cascade_plan.md`: cascade research and scraping blueprint using IS-BACKOFFICE pipeline
- `08_90_day_action_plan.md`: weekly action plan with owners, KPIs, and checkpoints
- `09_offer_pack_ingecart_inventory.md`: commercial packaging for the 3 fruit lines + 1 FFG
- `templates/prospect_pipeline.csv`: CRM starter template
- `templates/discovery_call_script.txt`: first-call script for demand diagnosis
- `templates/weekly_kpi_dashboard.csv`: weekly monitoring template

## How To Use This Pack In Practice
1. Complete technical data for all 4 available machines (`09_offer_pack_ingecart_inventory.md`).
2. Build first 120 target accounts from `03_icp_and_prospecting.md`.
3. Launch outbound cadence from `04_outreach_playbooks.md`.
4. Start scraping cascade in `07_scraping_cascade_plan.md` and refresh every 7 days.
5. Run 90-day execution plan in `08_90_day_action_plan.md`.

## Suggested Weekly Cadence
- Monday: update account intelligence + reprioritize pipeline
- Tuesday-Wednesday: outbound campaigns + discovery calls
- Thursday: proposals, video walkthroughs, and objection handling
- Friday: KPI review + plan corrections

## Success KPI Targets (first 90 days)
- 120 named target accounts loaded
- 300+ decision-maker contacts enriched
- 80+ qualified conversations
- 20+ opportunities in pipeline
- 6+ site-survey / technical validation meetings
- 2+ machine transactions closed or in advanced negotiation
