Turn current available inventory into fast-closing opportunities by packaging each machine with technical trust, execution confidence, and business ROI clarity.

## B. Mandatory Data Pack Per Machine (must complete)
- Manufacturer / model / year
- Installed location and running status
- Technical specs (speed, formats, board grades, print units, options)
- Condition report and maintenance history
- Video evidence (running cycle, quality output)
- Footprint and utility requirements
- Included accessories/spares
- Dismantling readiness and logistics constraints

## C. Sales Package Components
1. One-page machine sheet (visual + technical highlights)
2. 3-minute video walkthrough
3. Baseline performance statement (what is proven)
4. Scope options:
- EXW machine only
- Machine + dismantling + loading
- Turnkey relocation + installation + startup + training
5. Commercial terms:
- price
- staged payment milestones
- acceptance criteria
- warranty/support options

## D. ROI Framing Template
Use this simple equation in every proposal:

Annual Value = Labor Savings + Waste Reduction + Throughput Gain + Downtime Reduction - Service Cost

Payback (months) = Total Project Cost / (Annual Value / 12)

## E. Objection Handling Framework
- "Used machine risk" -> independent audit + acceptance protocol + startup support
- "Downtime fear" -> phased relocation plan + parallel preparation
- "Budget limitations" -> staged payment and phased scope
- "Capability doubts" -> references, test data, and technical Q&A session

## F. Priority Offers To Launch In First 30 Days
- Offer 1: "Fast Seasonal Capacity" (fruit machine + quick-start package)
- Offer 2: "Mid-Cap Converter Upgrade" (FFG + installation package)
- Offer 3: "Turnkey Relocation" (complete project management premium offer)

## G. Internal Rule
No machine goes to market without complete technical dossier and one validated ROI scenario for at least one target ICP.
